# Explore Wallpapers

A responsive wallpaper gallery website built using HTML, CSS, JavaScript, and Bootstrap 5.

## Features

* Responsive Design
* Wallpaper Gallery
* Search Functionality
* Download Buttons
* Categories Section
* About Page
* Contact Form
* Bootstrap 5 UI

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
